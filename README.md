### Module 1 - HTML5, CSS3, JavaScript

This module contains front-end development exercises covering:

* HTML5 Fundamentals
* CSS3 Styling
* Responsive Web Design
* JavaScript Basics
* DOM Manipulation
* Event Handling
* AJAX and Fetch API
* jQuery

### Module 1 - Bootstrap

This module contains Bootstrap 5 exercises covering:

* Bootstrap Setup
* Grid System
* Forms
* Buttons
* Navigation Components
* Cards
* Utilities
* Bootstrap Icons
* Bootstrap JavaScript Components

This module contains database exercises covering:

* SQL Queries
* Joins

* Views

This module contains Java programming exercises covering:

* Java Basics
* OOP Concepts
* Arrays and Collections
* Exception Handling
* File Handling
* JDBC
* Java Modules
* Advanced Java Concepts
